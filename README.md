# Raymond Membership & Points Demo

Multilingual, mobile-first membership and loyalty operations demo built with Next.js 16, React 19, Tailwind CSS 4 and Zustand.

## Included
- Dashboard and role-aware navigation
- Members, orders, points, redemption, reports, settings and permissions
- English, Simplified Chinese and Bahasa Melayu
- Raymond navy-and-gold responsive visual system

## Scope
Demo data is browser-persisted in local storage. This is not a production authentication, payment or customer-data backend.

## Validate

npm ci
npm run lint
npm run check
